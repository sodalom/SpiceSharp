Type Description
StateDerivative

A StateDerivative object that is compatible with this integration method.
Remarks

Tracked derivatives are used in more advanced features implemented by the integration method. For example, derived states can be used for finding a good time step by approximating the local truncation error (ie. the error made by taking discrete time steps). If you do not want the derivative to participate in these features, set track to false.
